# Microsoft Azure Data Engineering Assiciate(DP-203)

Start Installing the requirements.

```sh
# For Linux or WSL
python3 -m venv venv
source venv/bin/activate
pip install -r .\requirements.txt

# For Windows
python -m venv venv
.\venv\Scripts\activate
python.exe -m pip install --upgrade pip
pip install -r .\requirements.txt
```
